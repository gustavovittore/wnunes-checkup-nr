# Design Guide - Check-up NR Empresarial

## Objective
Create a premium, modern, corporate multi-step check-up interface for a Safety Work company.

## Style
- Clean, minimal, high-end
- Corporate and trustworthy
- Inspired by Apple, Stripe, Notion UI
- Strong typography
- Lots of white space

## Colors
- Primary: dark blue / navy
- Secondary: yellow or orange (alert tone)
- Background: white or light gray
- Text: dark gray / black

## Components
- Large clickable cards
- Smooth transitions
- Progress bar (step indicator)
- Centered layout
- Rounded corners (medium)
- Soft shadows

## Layout
- Centered container
- Max width around 600px
- Vertical flow
- Mobile-first design

## Interaction
- Each question in its own step
- Animated transitions between steps
- Highlight selected options
- Clear CTA buttons

## Tone
- Professional
- Clear
- Trustworthy
- Not aggressive
